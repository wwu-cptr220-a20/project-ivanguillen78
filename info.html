Potential Applications

Sharing Music Tastes and Exploring New Ones

The problem often lies in the fact that you are searching on your own. Imagine a web application that provides the opportunity to discover new music by interacting with other users.

Playlist Creator/Suggestion

Something similar to this? I'd be interested to see how this application creates playlists based on the song you input. I tested it a few times, and it was, for the most part, accurate in creating appropriate playlists.

Ideally, this application will have the capacity to:

Store user data

This includes being able to log in and view your account, set and change music preferences, and link other existing accounts for different applications such as Spotify (import/export playlists?), Instagram, and so on.

Support interaction between users

This is the basis for the entire application. Through interaction, users will be able to come across different music.

Listen to music (?)

Ideally, users should be able to listen to music while using the application.
